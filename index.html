About The Project

Using this web-app the user will be able to track their nutrition intake

Why this web-app?

Unique Data Input Methods

The web-app provides the user with multiple methods of data input such as:

  • Manual data entry
  • OCR based data input (Using Tesseract JS)
  • Automatic calculation of calories

Nutrition Intake Tracking Options

Users will be able to visually track their nutrition intake better with graphical representation of their data.

Report Generation

Users can export their nutrition intake data for external use in the form of a excel sheet or PDF
